# Setup

## 1. Create Supabase Project
- Create a free Supabase project
- Enable Email Auth (magic links)

## 2. Run Schema
- Open SQL Editor
- Paste `supabase_schema.sql`
- Execute

## 3. Configure Frontend
- Copy `.env.example`
- Insert your project URL and anon key

## 4. Deploy
- Upload repo to Netlify or Vercel
- Static deployment only required

## 5. Login
- Invite workshop users via Supabase Auth
- Use email magic links

## 6. Test
- Create transfer
- Verify realtime sync
- Verify current custodian updates
